# 项目默认文件介绍

## **main.cpp**

```c++
#include "mainwindow.h"
#include <QApplication> //应用程序类

//argc 命令行变量数量 argv 命令行变量数组

int main(int argc, char *argv[])
{

	//应用程序对象 在Qt中应用程序对象有且仅有一个
	QApplication a(argc, argv);

	//通过 窗口类实例化对象
	MainWindow w;

	//窗口不会默认弹出，需要调用show方法显示
	w.show();

	//a.exc();加入消息循环机制 阻塞功能
	return a.exec();

}
```
