# **向量积(矢积)与数量积(标积)的区别**

 

| 名称                              | 标积/内积/数量积/点积                     | 矢积/外积/向量积/叉积                                        |
| --------------------------------- | ----------------------------------------- | ------------------------------------------------------------ |
| 运算式（a，b和c粗体字，表示向量） | a·b=\|a\|\|b\|·cosθ                       | a×b=c，其中\|c\|=\|a\|\|b\|·sinθ，c的方向遵守右手定则        |
| 几何意义                          | 向量a在向量b方向上的投影与向量b的模的乘积 | c是垂直a、b所在平面，且以\|b\|·sinθ为高、\|a\|为底的平行四边形的面积 |
| 运算结果的区别                    | 标量（常用于物理）/数量（常用于数学）     | 矢量（常用于物理）/向量（常用于数学）                        |
